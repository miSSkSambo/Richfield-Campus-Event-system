# 🎓 Campus Event Management System (PROGRAMMING 731)

## 📌 Overview
This project is a Java console-based application developed for the Programming 731 module.  
The system allows staff and students to manage and participate in campus events with features such as event creation, registration, waitlists, and data persistence.

---

## 🚀 Features
- Role-based access (Staff & Student)
- Event creation and management
- Event registration and waitlist system
- Automatic waitlist promotion
- Search functionality
- Input validation and error handling
- File-based data persistence
